# بوابة العضو (Member Portal)

## الفلسفة
المشروع من الداخل معقد، لكن **الواجهة لا يجب أن تكون معقدة**. العضو العادي لا يرى نموذج البيانات أو المحركات. يرى فقط ما يخصه بلغة بسيطة.

---

## الشاشات الأساسية للعضو

### 1. لوحة القيادة الشخصية
- ملخص سريع: عدد الصناديق، إجمالي الالتزام الشهري، إشعارات معلقة.
- تنبيهات التكرار (إذا كان يدفع لنفس الغرض في كيانين).

### 2. مشاركاتي
- قائمة الكيانات التي أنا عضو فيها.
- لكل كيان: المحافظ المشترك فيها والمسار المختار.

### 3. التزاماتي
- إجمالي ما أدفعه شهرياً **عبر كل الكيانات**.
- تفصيل لكل محفظة: المبلغ، تاريخ الاستحقاق، حالة الدفع.

### 4. حقوقي
- البنود التي يحق لي الاستفادة منها.
- شروط الاستفادة لكل بند.
- المدة المتبقية قبل الأهلية (إن وجدت).

### 5. التصويتات
- التصويتات المطلوبة مني حالياً.
- نتائج التصويتات السابقة التي شاركت فيها.

### 6. طلباتي
- طلبات الصرف التي قدمتها.
- حالة كل طلب (قيد المراجعة، مقبول، مرفوض).

### 7. اعتراضاتي
- الاعتراضات التي رفعتها.
- حالة كل اعتراض ورد الجهة المعنية.

### 8. المحافظ المقترحة لي
- محافظ متوافقة مع شروطي ولم أشترك فيها بعد.
- نسبة التوافق (مثلاً: "متوافقة مع شروطك بنسبة 92%").

### 9. المحافظ غير المتوافقة
- محافظ لا تطابق شروطي، مع توضيح سبب عدم التوافق.

---

## مثال لما يراه العضو

```
┌─────────────────────────────────────────────┐
│  محفظة الطوارئ — متوافقة مع شروطك (92%)    │
│                                             │
│  الاشتراك: 30 ريال / شهر                    │
│  الحوكمة: لجنة + مراجع                      │
│  حقك: الاستفادة بعد 90 يوماً                │
│                                             │
│  [اشترك]   [لا أشترك]   [أشارك بشرط]       │
└─────────────────────────────────────────────┘
```

## نموذج حالة العضو (Member State Machine)

كل اشتراك للعضو داخل مسار له حالة:

```
Interested → Conditional → Active → Suspended → Exited
                                ↘ SupporterOnly
```

| الحالة | المعنى |
|---|---|
| `Interested` | مهتم — لم يقرر بعد |
| `Conditional` | يريد الاشتراك إذا تحققت شروطه |
| `Active` | متوافق ويدفع ويستفيد |
| `Suspended` | توقف بسبب عدم الدفع أو تغيير الشروط |
| `Exited` | خرج من المسار |
| `SupporterOnly` | داعم بلا حق استفادة |

> لتفاصيل حماية العضو عند تغيير الشروط، انظر: [member_participation_contract.md](../05_Rules_and_Governance/member_participation_contract.md)
