# إدارة الاعتراضات والنزاعات (Dispute Management)

## المبدأ
كل عضو له الحق في الاعتراض على أي قرار يمس حقوقه أو ماله. النظام يوفر مسار اعتراض رسمي وموثق، يحمي العضو من التعسف ويحمي الإدارة من الاتهامات غير الموثقة.

---

## أنواع الإجراءات

| النوع | المعنى | الخطورة |
|---|---|---|
| طلب توضيح | العضو يطلب شرح قرار أو عملية | منخفضة |
| اعتراض | العضو يرفض قراراً ويطلب مراجعته | متوسطة |
| مراجعة | طلب تدقيق رسمي من المراجع | متوسطة |
| تصعيد | نقل الاعتراض لجهة أعلى | عالية |
| نزاع داخلي | خلاف بين أعضاء أو بين عضو والإدارة | عالية |
| شبهة مخالفة لائحة | اتهام بمخالفة القواعد المعتمدة | عالية |
| شبهة قانونية | اتهام بمخالفة قانونية تحتاج جهة خارجية | حرجة |

---

## نموذج الاعتراض (Appeal Model)

```
Appeal {
  againstDecision   -- القرار المعترض عليه
  reason            -- سبب الاعتراض
  evidence          -- المستندات الداعمة
  requestedAction   -- ما يطلبه المعترض
  reviewer          -- الجهة المراجعة
  status            -- مفتوح / قيد المراجعة / مغلق / مصعّد
  timeline          -- المدة الزمنية للرد
}
```

---

## أسباب الاعتراض

| السبب | مثال |
|---|---|
| مخالفة اللائحة | تم الصرف بدون تصويت مع أن اللائحة تشترط التصويت |
| تعارض مصالح | المدير وافق على صرف لنفسه أو قريبه |
| نقص مستندات | تم الصرف بدون فاتورة أو تقرير طبي |
| تجاوز صلاحية | المدير صرف مبلغاً أعلى من صلاحيته |
| صرف من محفظة خاطئة | تم استخدام مال الطوارئ لصيانة المصعد |
| مخالفة شروط العضو | تم تغيير الحوكمة بدون إشعار العضو |
| تصويت غير صحيح | تم احتساب أصوات أعضاء غير مؤهلين |
| طلب إيضاح | العضو لا يفهم سبب قرار معين |

---

## دورة حياة الاعتراض

```
تقديم الاعتراض
  ↓
مراجعة أولية (هل مكتمل؟ هل مؤهل؟)
  ↓
إحالة للجهة المعنية (لجنة / مراجع / مجلس إدارة)
  ↓
رد الجهة (قبول / رفض / تعديل / طلب مستندات)
  ↓
إبلاغ المعترض
  ↓
قرار نهائي أو تصعيد
```

---

## قواعد مهمة

1. **لكل اعتراض مدة زمنية للرد** — إذا لم تردّ الجهة المعنية خلال المدة، يُصعّد تلقائياً.

2. **المعترض لا يُعاقب على الاعتراض** — طالما أنه مبني على سبب واضح ومستندات.

3. **في حال النزاع:** يرجع محرك النزاعات إلى **النسخة التاريخية** من اللائحة وقت حدوث القرار، وليس النسخة الحالية.

4. **التصعيد القانوني:** النظام يوثق كل شيء بشكل يسهّل الرجوع إليه في حال احتاج الأمر جهة خارجية (محامٍ أو جهة رسمية).
