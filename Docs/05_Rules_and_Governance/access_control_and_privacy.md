# الصلاحيات والشفافية والخصوصية (Access Control & Privacy)

> هذا الملف يصف PBAC **داخل الكيان** — أي بين مستخدمي الكيان الواحد.

## المبدأ: صلاحيات مبنية على السياسة والسياق

النظام يستخدم **Policy-Based Access Control (PBAC)** وليس فقط Role-Based Access Control (RBAC). لأن الدور وحده لا يكفي لتحديد ما يراه العضو وما يفعله.

---

## عوامل تحديد الصلاحية

الصلاحية لا تعتمد فقط على "هل هو عضو أو مدير"، بل على مجموعة عوامل مجتمعة:

| العامل | مثال |
|---|---|
| عضو في أي كيان؟ | كيان العائلة فقط، أو العائلة والعمارة |
| مشترك في أي محفظة؟ | محفظة الطوارئ، محفظة الخدمات |
| في أي مسار؟ | مسار مجلس الإدارة، مسار القرار الفردي |
| هل دفع؟ | مشترك فعال أو متأخر |
| هل هو مستفيد؟ | له حق الاستفادة أو داعم فقط |
| هل هو مراجع؟ | دور المراجع يمنح صلاحيات اطلاع إضافية |
| هل البند حساس؟ | بند علاج يحتاج خصوصية أعلى |
| هل البيانات مالية أم شخصية؟ | المال يُكشف، الهوية قد تُخفى |
| هل القرار يخص ماله؟ | صلاحيات إضافية إذا كان المال ماله |

---

## أمثلة عملية

**مثال 1:**
```
العضو يرى: إجمالي صرف بند العلاج (5,000 ريال)
العضو لا يرى: اسم المستفيد
إلا إذا كان: في لجنة التكافل
```

**مثال 2:**
```
العضو يصوت: في مسار مجلس الإدارة
العضو لا يصوت: في مسار القرار الفردي
السبب: ليس مشتركاً فيه
```

## مستويات الشفافية

كل معلومة في النظام لها مستوى ظهور محدد:

| المستوى | المعنى |
|---|---|
| `PublicToMembers` | متاح لكل أعضاء الكيان |
| `VisibleToParticipants` | متاح لمشتركي المحفظة أو المسار فقط |
| `VisibleToCommittee` | متاح للجنة فقط |
| `VisibleToAuditor` | متاح للمراجع فقط |
| `HiddenSensitive` | مخفي تماماً إلا لمعالج الحالة |
| `AggregatedOnly` | يظهر كإجمالي فقط بدون تفاصيل أفراد |

---

## تطبيق الشفافية على أنواع المعلومات

| نوع المعلومة | من يراها؟ |
|---|---|
| إجمالي رصيد المحفظة | المشتركون |
| مبلغ الصرف | المشتركون |
| اسم المستفيد في حالة علاج | اللجنة فقط |
| التقرير الطبي | اللجنة والمراجع فقط |
| الفاتورة | المراجع وربما اللجنة |
| سبب عام للصرف | الأعضاء |
| نتيجة التصويت | حسب سياسة التصويت |
| أسماء المصوّتين | علني أو سري حسب نوع القرار |

---

## القاعدة الذهبية للشفافية

> **نخفي الإنسان عند الحاجة، ولا نخفي المال.**

يعني:
- المبالغ المالية دائماً شفافة لأصحاب الحق.
- هوية المستفيد قد تُخفى في الحالات الحساسة (علاج، مساعدة، دعم).
- لا يوجد صرف مخفي — كل ريال يظهر في التقارير ولو بدون ذكر الاسم.
