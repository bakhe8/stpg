# مراحل التنفيذ (Implementation Phases)

المشروع كبير ومعقد، لكنه يُبنى بالتدرج. كل نسخة تضيف طبقة جديدة من القدرات دون كسر ما قبلها.

---

## النسخة 1 — الأساس (Foundation)
**الهدف:** صندوق بسيط يعمل بالكامل لجماعة واحدة.

| المكون | الوصف |
|---|---|
| كيان واحد | إنشاء صندوق (عائلة / عمارة / جيران) |
| أعضاء | تسجيل الأعضاء وإدارة عضوياتهم |
| محافظ | إنشاء محفظة واحدة أو أكثر (طوارئ، خدمات) |
| اشتراكات | التزام شهري ثابت مع تتبع الدفع |
| صرف | طلب صرف + موافقة المسؤول |
| تقارير | تقرير شهري مالي بسيط |
| تصويت بسيط | موافقة / رفض على القرارات |
| اعتراض بسيط | تقديم اعتراض مكتوب مع رد المسؤول |

> **مستوى التشغيل:** البسيط — كيان + محفظة + مسؤول واحد + تقرير شهري.

---

## النسخة 2 — الحوكمة (Governance)
**الهدف:** إضافة مسارات الحوكمة وشروط العضو، وهي الميزة الجوهرية في المشروع.

| المكون | الوصف |
|---|---|
| شروط العضو | تفضيلات العضو (نوع الحوكمة المقبولة، سقف الصرف) |
| مسارات الحوكمة | تعدد مسارات الثقة داخل المحفظة الواحدة |
| بنود الصرف | تصنيف الصرف ببنود لها شروط ومستندات وسقف |
| محرك التوافق | مقارنة شروط العضو مع خصائص المسار |
| حالات العضو | Interested → Conditional → Active → Suspended → Exited |
| عقد المشاركة | Snapshot للشروط وقت الاشتراك |

> **مستوى التشغيل:** المنظم — عدة محافظ + لجنة + تصويت + اعتراضات + بنود صرف.

---

## النسخة 3 — العلاقات (Relationships)
**الهدف:** دعم تعدد الكيانات والعلاقات بينها.

| المكون | الوصف |
|---|---|
| تعدد الكيانات | العضو مشترك في أكثر من صندوق |
| علاقات بين كيانات | تداخل، دعم، تفويض، دمج |
| محافظ مشتركة | محفظة يمولها كيانان أو أكثر |
| تحليل التداخل | كشف تكرار الدفع والتداخل بين الصناديق |
| منع خلط الأموال | دفتر مالي مقسم لكل كيان ومسار |

> **مستوى التشغيل:** المتقدم — عدة كيانات + علاقات + محافظ مشتركة.

---

## النسخة 4 — الذكاء والنضج (Intelligence)
**الهدف:** أدوات تحليل متقدمة وحل النزاعات.

| المكون | الوصف |
|---|---|
| نزاعات متقدمة | تصعيد، مراجعة، قرار نهائي |
| مراجعة قانونية | ربط بالأنظمة واللوائح |
| محرك توصيات | اقتراح أفضل نموذج حوكمة بناءً على بيانات الأعضاء |
| مركز صحة الصندوق | إنذار مبكر: إرهاق دفع، انخفاض مشاركة، محافظ ميتة |
